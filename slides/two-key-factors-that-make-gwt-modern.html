Two key factors that will make GWT modern

First factor

  • A bleezing fast SuperDevMode
  • that can transpile single file in milliseconds
  • allowing us to use Java as a browser language

Second factor

  • The so called JsInterop
  • A typed interoperability between Java and JavaScript that will allow us to:
    • export Java API to JavaScript
    • define strongly-typed interfaces to JavaScript objects

 

All done in pure Java, interfaces, classes and annotations

Third factor

Yes, we said two, but GSS, the new API to use Google Closure StyleSheet just inside GWT, it's really promising
