What is Modern Web Development ?

It's a flexible and comprehensive JavaScript Toolchain:

  • Node Package Manager
  • Bower
  • Grunt

An agile development lifecycle

  • Automatic Dependency Management
  • Generator available for a lot of frameworks, for a quick startup
  • Development Web Server with automatic reload

The sensation of not wasting time...

  • Compiling classes
  • Generating JavaScript
  • Building an archive

While you want to be creative and effective !!!
